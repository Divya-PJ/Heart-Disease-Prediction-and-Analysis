# Heart Disease Prediction and Analysis
## Cardiovascular Disease Prediction using Machine Learning and Tableau Dashboard Visualization.

## Project Overview

Cardiovascular diseases are among the leading causes of death worldwide. This project focuses on analyzing cardiovascular disease risk factors and developing a machine learning model to predict heart disease occurrence.

The project combines:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Statistical Analysis
* Logistic Regression Modelling
* Tableau Dashboard Visualization

---

## Dataset Information

The dataset contains patient health records with 14 clinical attributes including:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Maximum Heart Rate
* Exercise-Induced Angina
* Thalassemia
* Heart Disease Status

After data cleaning:

* Records: 302
* Features: 14
* Missing Values: 0
* Duplicate Records Removed: 1

---

## Exploratory Data Analysis

The analysis investigated:

* Occurrence of heart disease across age groups
* Relationship between cholesterol and heart disease
* Resting blood pressure analysis
* Gender composition
* Thalassemia impact on cardiovascular disease
* Peak exercise effects on disease occurrence

---

## Machine Learning Model

Algorithm Used:

* Logistic Regression

Model Performance:

* Accuracy: 78.69%
* ROC-AUC Score: 0.8647

Classification Report:

* Precision (Healthy): 0.83
* Recall (Healthy): 0.68
* Precision (Diseased): 0.76
* Recall (Diseased): 0.88

## Key Findings

* Heart disease occurrence is highest among individuals aged 50–60 years.
* Chest pain type and thalassemia are significant predictors.
* Exercise-related variables show strong relationships with cardiovascular disease.
* Logistic Regression demonstrated strong predictive capability for disease detection.
